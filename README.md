# DeltaSigma Shop - Backend

## Prerequisites
- Java JDK 17+ installed
- Maven or Gradle (depending on your project setup)
- An IDE like IntelliJ IDEA, Eclipse, or VS Code (optional)

## Installation & Running Locally

1. **Clone the repository**

```bash
git clone https://github.com/dadslittleprincess09/deltasigmaproject.git
cd deltasigmaproject

2.Build the project
If using Maven : mvn clean install

3.Run the application
Using Maven : mvn spring-boot:run
